🎮 Maze Game

⏱️ Time: 0s

📋 How to Play:

  • 🎮 Use Arrow Keys, WASD, or D-Pad Buttons to move
  • 🟢 Green = Start position
  • 🔴 Red = Goal (reach to win)
  • 🟦 Blue = Your player
  • ⏱️ Timer tracks your completion time
  • 🔄 Use Reset Game button to restart the current level

🛠️ Customization:

To add or modify mazes:

  1. Open mazes.js
  2. Add your maze to the MAZES array
  3. Use: 1=wall, 0=path, 2=start, 3=end
  4. Save and refresh the page
